Latte Art

  • Spout Proximity The further up you are, the more swaying and swirling happens, blurring the design.

  • Starting Point Don’t start in the center, start higher. The more drops necessary, the deeper the start can be because the ripples will move around.

  • Pushing Forward Move backward as little as possible, only necessary if the design requires it.
