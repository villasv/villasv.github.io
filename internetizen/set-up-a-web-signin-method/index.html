Set Up a Web Sign-In Method

If you already Use a Custom Email Address, a simple method is to use email authentication supported by RelMeAuth.

RelMeAuth might support one of the platforms or social networks you have an account, but consider a bonus independence point not relying on those to authenticate you anywhere.

Additionally, configuring IndieAuth with IndieAuth.com for authorization will allow you to use your email-based RelMeAuth to sign in to websites that don’t support RelMeAuth directly but do support the IndieAuth standard.
