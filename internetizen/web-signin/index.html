Web Sign-In

Web Sign-In is an authentication method where the identity is based on a web address. A client that supports Web Sign-In will then inspect the given address to find an authorization method to use as next step.

One common implementation of Web Sign-In is RelMeAuth, which uses rel=me links to find a service to which authentication can be delegated. Another option is IndieAuth, where the client negotiates with a specified authorization endpoint.
